# Mission Control

Visual command center for AI bot management.

## Features

- **Bot Squad** - Visual overview of all active bots (Goober, Code Bot, Research Bot, Writer Bot)
- **Approval Queue** - Review and approve/reject tasks before they go live
- **Suggestions** - AI-generated recommendations for projects and improvements
- **Activity Log** - Real-time track of what each bot is working on

## Access

Visit: https://kendellproduction.github.io/mission-control/

## Architecture

- **Goober** (👑) - Command bot, manages other bots, interfaces with you
- **Code Bot** (💻) - Handles all coding tasks
- **Research Bot** (🔍) - Competitor analysis, grants research, market research
- **Writer Bot️) - Marketing** (✍ copy, content creation
